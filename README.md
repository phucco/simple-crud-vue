# simple-crud-vue

Simple CRUD app in single page with Vue 3

- Add new posts
- Edit posts
- Delete posts

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
